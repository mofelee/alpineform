## Summary

- This prerelease is incomplete and must not be used. The signed tag and partial
  assets remain published for auditability.
- AlpineForm provides a reviewable validate, plan, apply, and check loop for
  persistent Alpine 3.24 hosts.
- Real Alpine 3.24.1 x86_64 VM gates cover the complete Beta core.
- Four static CLI archives, checksums, a Sigstore checksum bundle, and four
  SPDX SBOMs were published before GitHub rejected provenance persistence.

## Compatibility

- Release phase: alpha; stable compatibility is not promised.
- CLI artifacts: Linux and macOS on amd64 and arm64.
- Beta managed target: Alpine 3.24 x86_64.
- Preview managed target: Alpine 3.24 aarch64; no blocking real-VM gate.
- This release supersedes the incomplete `v0.1.0-alpha.1` prerelease. There is
  no AlpineForm state migration.

## Breaking Changes

- None relative to the alpha.1 code or state format.

## Migration Notes

- Do not use alpha.1 release artifacts. Install alpha.2 directly.
- Back up `/var/lib/alpineform/state.json` before later alpha upgrades.

## Added

- Alpine fact discovery, root SSH, atomic state, renewable leases, deterministic
  plans, native Alpine resources, verified prebuilt components, and deduplicated
  change scripts.
- Blocking no-op, drift, repair, lifecycle, secret, and reboot VM assertions.
- Signed release checksums, per-archive SPDX SBOMs, and the checksum-verifying
  installer. GitHub attestations did not complete.

## Changed

- Alpha.1 and alpha.2 tags and partial assets remain available only as auditable
  failed releases.

## Fixed

- Install Cosign under the command name invoked by the release workflow so
  signing and all downstream verification can run.

## Security

- Protected values are omitted from plan, graph, state, HTML, debug, errors, and
  uploaded integration diagnostics.
- Component artifacts and release archives are checksum verified before use.

## Known Issues

- Alpine 3.24 aarch64 target support is Preview because it lacks a real-VM gate.
- Custom APK signing keys and CA-certificate components have unit/provider
  coverage but are not in the blocking VM fixture.
- Homebrew is not published for this alpha.
- GitHub artifact attestations are unavailable to this user-owned private
  repository. Downstream Linux/macOS installer and fresh-Alpine verification
  jobs were skipped after provenance persistence failed.
- Docker (#10), rollback-safe nftables (#13), and target-side source builds
  (#14) remain independent follow-ups outside the v0.1 core.
